# Stage 1：プログラミング入門

**目安：第 1〜3 週（各週 約 10 時間）**

Stage 1 では、**JavaScript** を **Node.js** で動かしながら、プログラミングの基礎（変数・分岐・ループ・関数）を学びます。最後に、小さな CLI ツールを 1 つ完成させることが目標です。

- **Node.js** は、[受講開始前の準備](/preparation)でインストールしたものをそのまま使います。まだ準備が終わっていない場合は、先に [準備教材](/preparation) を読んで、エディタ・ターミナル・Node.js の環境を整えてから始めてください。

---

## Stage 1 修了チェック

Stage 1 を修了したかどうかは、次のチェックリストで確認できます。すべて「はい」なら修了です。

- [ ] 変数（`let` / `const`）を使って、数値・文字列・真偽値を扱える
- [ ] `if` / `else if` / `else` で条件に応じた分岐を書ける
- [ ] `for` や `while` でループを書ける
- [ ] 関数を定義し、引数と戻り値を使って処理をまとめられる
- [ ] 配列を作成・アクセスし、ループと組み合わせて使える
- [ ] 第 3 週の CLI ツール課題を完成させ、`node ファイル名.js` で実行できる
- [ ] コマンドライン引数（`process.argv`）を読んで、プログラムの動作を変えられる

（公式な提出物評価は Stage 2 以降で行います。Stage 1 は「修了基準の達成」で扱います。詳細は [ルーブリック](/rubric) を参照。）

---

## Stage 2 への接続

Stage 2 では **Git**（clone, commit, push, branch, merge）を学びます。そのとき、**Stage 1 で作った CLI ツールをリポジトリに commit する**と、自然に「コードをバージョン管理する」体験ができます。第 3 週で完成させたプログラムは、そのまま Stage 2 の最初の commit の対象にするとよいです。
