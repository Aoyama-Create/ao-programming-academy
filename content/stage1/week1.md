
### REPL とファイル実行の違い

Node には 2 つの主な使い方があります。

1. **REPL（対話モード）**: ターミナルで `node` とだけ打って Enter すると、1 行ずつ JavaScript を入力してすぐ結果を見られます。動作確認やちょっとした計算に向いています。
2. **ファイル実行**: エディタで書いた `.js` ファイルを、`node ファイル名.js` でまとめて実行します。本格的なプログラムはこちらで書きます。

第 1 週では、まず REPL で少し試してから、`.js` ファイルを作って実行する練習をします。

## 1. 変数と型

### 変数とは

**変数**は、データを入れておく「箱」のようなものです。名前を付けて、あとから中身を参照したり変更したりできます。

JavaScript では、変数を宣言するときに `let` または `const` を使います。

- `let`: あとから値を変更できる変数
- `const`: 再代入しない（定数のような）変数。推奨は「まず `const` で宣言し、どうしても変えたいときだけ `let`」です。

```javascript
const name = "太郎";
let count = 0;
count = count + 1; // count は 1 になる
```

### 主な型（プリミティブ）

| 型 | 例 | 説明 |
|----|-----|------|
| 数値（number） | `42`, `3.14` | 整数や小数 |
| 文字列（string） | `"こんにちは"`, `'Hello'` | 引用符で囲んだ文字の並び |
| 真偽値（boolean） | `true`, `false` | 真か偽か |

```javascript
const age = 16;
const message = "歳です";
const isStudent = true;
```

## 2. 演算子

### 算術演算子

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `+` | 足し算（文字列の連結も） | `3 + 5` → `8` |
| `-` | 引き算 | `10 - 4` → `6` |
| `*` | かけ算 | `2 * 6` → `12` |
| `/` | 割り算 | `10 / 4` → `2.5` |
| `%` | 余り | `10 % 4` → `2` |

### 比較演算子

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `===` | 等しい | `3 === 3` → `true` |
| `!==` | 等しくない | `3 !== 5` → `true` |
| `<`, `>`, `<=`, `>=` | 大小 | `5 > 3` → `true` |

### 論理演算子

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `&&` | かつ | `true && false` → `false` |
| `\|\|` | または | `true \|\| false` → `true` |
| `!` | 否定 | `!false` → `true` |

## 3. 分岐（if）

条件に応じて、実行する処理を変えたいときに `if` を使います。

```javascript
const score = 80;

if (score >= 60) {
  console.log("合格です");
} else {
  console.log("不合格です");
}
```

複数の条件を並べる場合は `else if` を使います。

```javascript
const temperature = 25;

if (temperature >= 30) {
  console.log("暑いです");
} else if (temperature >= 20) {
  console.log("過ごしやすいです");
} else {
  console.log("寒いです");
}
```

`console.log(...)` は、ターミナルに文字などを出力するための関数です。実行結果を確認するときに使います。

## 4. ループ（for / while）

### for（回数が分かっているとき）

「〇回繰り返す」というときに `for` が便利です。

```javascript
for (let i = 0; i < 5; i++) {
  console.log(i); // 0, 1, 2, 3, 4 が順に出力される
}
```

- `let i = 0`: 最初に `i` を 0 にする
- `i < 5`: `i` が 5 未満のあいだだけ繰り返す
- `i++`: 1 回ごとに `i` を 1 増やす

### while（条件が満たされているあいだ繰り返す）

「〇の間ずっと繰り返す」というときに `while` を使います。

```javascript
let n = 0;
while (n < 3) {
  console.log(n);
  n++;
}
```

条件を間違えると**無限ループ**（終わらなくなる）になるので注意してください。

## 6. よくあるエラーの例

### 未定義の変数を使う

```javascript
console.log(unknownVariable);
```

`unknownVariable` を宣言していないため、実行すると「ReferenceError: unknownVariable is not defined」のようなエラーになります。変数は、使う前に `let` や `const` で宣言しましょう。

### 型の勘違い

```javascript
const a = "5";
const b = 3;
console.log(a + b); // "53" になる（文字列の連結）
```

`a` が文字列なので、`+` は「数値の足し算」ではなく「文字列の連結」になります。数値として計算したい場合は、`Number(a)` で数値に変換するなどの工夫が必要です。

## 7. 演習課題

以下の課題に取り組みましょう。各問で「条件」「期待する出力」「ヒント」を参考に、`.js` ファイルを作って `node ファイル名.js` で実行して確認してください。

### 課題 1：偶数判定

**条件**: 変数 `n` に数値を入れ、その数が**偶数なら**「偶数です」、**奇数なら**「奇数です」と表示するプログラムを書きなさい。

**期待する出力**: `n` が 4 のとき「偶数です」、`n` が 7 のとき「奇数です」。

**ヒント**: 偶数かどうかは `n % 2 === 0` で判定できます。

---

### 課題 2：1 から N までの合計

**条件**: 変数 `n`（例：5）に数値を入れ、1 から n までの整数の合計を計算して表示するプログラムを書きなさい。ループ（`for` または `while`）を使うこと。

**期待する出力**: `n` が 5 のとき、合計は 1+2+3+4+5 = 15 なので「15」と表示される。

**ヒント**: 合計を入れる変数（例：`let sum = 0`）を用意し、ループで 1, 2, 3, ... を順に足していきます。

---

### 課題 3：FizzBuzz の一部（1〜15）

**条件**: 1 から 15 まで、数を順に表示する。ただし、**3 の倍数のときは数字の代わりに「Fizz」**、**5 の倍数のときは「Buzz」**、**3 と 5 の両方の倍数のときは「FizzBuzz」** と表示する。それ以外はそのまま数字を表示する。

**期待する出力**:  
`1`, `2`, `Fizz`, `4`, `Buzz`, `Fizz`, `7`, `8`, `Fizz`, `Buzz`, `11`, `Fizz`, `13`, `14`, `FizzBuzz`

**ヒント**: `for` で 1 から 15 まで回し、`if` / `else if` / `else` で「3 と 5 の公倍数」「3 の倍数」「5 の倍数」「それ以外」に分岐させます。

---

### 課題 4：九九の一段目

**条件**: 変数 `k`（例：3）に 1〜9 の数値を入れ、**その段の九九**（3 の段なら 3×1, 3×2, ..., 3×9）を順に表示するプログラムを書きなさい。`for` を使うこと。

**期待する出力**: `k` が 3 のとき、`3`, `6`, `9`, `12`, ... , `27` が順に表示される。

**ヒント**: `for (let i = 1; i <= 9; i++)` で 1 から 9 まで回し、`k * i` を表示します。

---

### 課題 5：小さい方の数

**条件**: 変数 `a` と `b` に数値を入れ、**小さい方の数**を表示するプログラムを書きなさい。`if` と `else` を使うこと。

**期待する出力**: `a` が 10、`b` が 7 のとき「7」と表示される。

**ヒント**: `if (a < b)` なら `a` を、そうでなければ `b` を表示します。
