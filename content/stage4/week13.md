## 1. Laravel と MVC

Laravel は PHP の Web フレームワークです。**Model（モデル）**・**View（ビュー）**・**Controller（コントローラ）** の役割に沿ってコードを配置します。

- **Model**: データ（DB テーブル）とのやりとり、ビジネスロジックの一部
- **View**: 画面表示（API の場合は JSON レスポンスを返す部分もコントローラ側で扱うことが多い）
- **Controller**: リクエストを受け取り、Model を呼び出し、レスポンスを返す

## 3. Eloquent とマイグレーション

**Eloquent** は Laravel の ORM です。モデルクラスとテーブルを対応づけ、`User::find(1)` のようにオブジェクトとして扱えます。

**マイグレーション**は、DB のスキーマを PHP のファイルで管理する仕組みです。`php artisan migrate` でテーブルを作成・更新し、チームや環境間で再現できます。
